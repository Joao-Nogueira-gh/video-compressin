encode_video function

[in]limitFramesOptional parameter for limiting number of frames to encode Starts by encoding the header, passing additional parameters such as the Golomb factor Uses intra-coding method in the first frame, as described in the IntraCodec class Uses inter-coding for all the remaining frames That is by constructing a matrix of blocks for every frame, finding the most similar block of the previous frame to each one, and encoding that block of errors and the vector related to the most similar block's position
